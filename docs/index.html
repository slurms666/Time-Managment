Keep your screen awake for legitimate productivity tasks

This browser-based utility simulates small, natural pointer movement while active so your system is less likely to sleep during presentations, screen sharing, monitoring, long downloads, long-form reading, or remote sessions.

No install required. Lightweight and straightforward.

How it works

When active, the tool performs subtle pointer movement at your selected interval. You can start and stop at any time.

Legitimate use cases

  • Presentations and screen sharing
  • Monitoring dashboards and logs
  • Long downloads and software installs
  • Long-form reading and documentation review
  • Remote desktop sessions with inactivity timeouts
  • Accessibility situations where repetitive input is difficult

Features

  • Start/Stop controls
  • Active/Inactive status indicator
  • Adjustable interval
  • Lightweight browser-based operation

Privacy

Runs locally in your browser. No analytics or tracking libraries are included in this site.

Responsible use

This tool is intended for legitimate screen-awake workflows only. It must not be used to evade, defeat, or misrepresent activity in workplace productivity or monitoring systems. Always follow your employer’s policies and local regulations.
