Time Management
Browser utility for screen-awake workflows

Keep your screen awake during focused work

Updated 2026 • Browser-based utility • Local operation

This tool helps reduce interruptions from idle sleep during legitimate productivity tasks. While active, it simulates small pointer movement and periodic clicks inside the browser window.

How to use: Click Start Screen Awake to activate movement. Use Interval to control how often movement occurs. Toggle Focus View switches to a plain 404-style cover screen (Esc to exit). Click Stop to fully deactivate.

Legitimate use cases

  • Presentations and screen sharing sessions
  • Monitoring dashboards and log consoles
  • Long downloads and software installs
  • Long-form reading and documentation review
  • Remote desktop sessions that disconnect on inactivity
  • Accessibility situations where repeated input is difficult

Features

  • Start/Stop controls
  • Active/Inactive status indicator
  • Adjustable interval
  • Lightweight browser-based operation

Privacy

Runs locally in your browser. No analytics libraries are included.

Responsible use

This utility is for legitimate productivity scenarios only. Do not use it to bypass, undermine, or misrepresent activity to workplace monitoring or productivity systems. Use it in line with your organization’s policies.

FAQ

Does this install anything? No, it runs in the browser.

Can I change how often it runs? Yes, use the interval slider.

Does it collect data? No tracking or analytics scripts are included.
